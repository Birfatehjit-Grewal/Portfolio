SQLite Library Database Application

Jul 2023 - Aug 2023

Languages: Python and SQLite

Applications: IDLE(Python 3.9 64 bits) and Jupyter Notebook

Description of project:

This is a command line app which simulates a library database. The users can use this to find items like CDs or books that are currently in stock or find information about the availablity of items. The user can then borrow the item which will mark that item as unavailable. The application also allows the user to reserve a event room and register for events. Information like the availablity of items, who has borrowed which item and what rooms have been booked are stored in a SQLite database. This database is queried for the app features like checking the users currently borrowed items and returning items to the library.

Word Sudoku Android App

Sep 2023 - Dec 2022

Languages: Java

Applications: Android Studio and Git

Description of project:

I developed a Word Sudoku App using Android Studio which helps learn a second language through a sudoku game. The app includes different size grids and 2 modes. The modes signify if the board will have numbers and will pronunce the word when clicked or if the board will have french words on it and the user will need to translate them to solve the Sudoku. The development process involved using Git for version controll and JUint was used to create unit tests to test the functions of the app. Moreover, manual testing was done to test the user interface.

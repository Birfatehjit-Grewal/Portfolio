Image feature Matching

Mar 2025

Programming Languages: Python

Applications: VScode

The Source Code is not public due to SFU policy

Project Description:

The program uses FAST detectors to get key points in images and creates BRIEF descriptors for each point. The keypoints are then compared with the keypoints in a second image using the descriptors to match points. The matched points are used to compute a homography matrix using RANSAC and normilization. The matrix can then be used to map another image on to the instance of the object in the first image.

RDT Protocol

Nov 2024

Programming Languages: Python

Applications: VScode

Source Code
Project Description:

This protocol implements a connection-oriented, reliable, pipelined data transfer protocol which includes flow control and congestion control mechanisms. The protocol using UDP sockets as a base and add features like 3-way hand handshake, Loss recovery through retransmission with a timeout and fast retransmission using duplicate ACKs. I also implemented a checksum in case of bit flips. The flow control is handled by reading the available buffer space in the response and limiting the sent packets to avoid overflow. I use slow-start congestion control as my congestion control mechanism.

Applications: VScode

Source Code
Project Description:

This project includes 2 servers created using socket programming. The servers implemented basic HTTP Methods and messages. The servers can produce the following codes 200 OK, 304 Not Modified, 400 Bad Request, 404 Not Found, and 501 Not Implemented. The response is based on the RFC7231. The proxy server reduces the workload of the main server by caching the response for get requests by requesting the document from the main server as a conditional get request based on the cached file already in the proxy server.
